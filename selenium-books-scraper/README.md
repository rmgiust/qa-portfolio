# 📚 Selenium Books Scraper (TypeScript)

## 🇧🇷 Sobre

Script de automação utilizando Selenium + TypeScript para extrair títulos e preços de livros de um site público.

Este projeto demonstra conceitos fundamentais de QA, como interação com DOM, sincronização (waits) e extração de dados.

---

## 🇺🇸 Overview

Automation script built with Selenium WebDriver and TypeScript to scrape book titles and prices from a public website.

---

## 🧰 Tecnologias | Tech Stack

* Selenium WebDriver
* TypeScript
* Node.js

---

## ⚙️ Como rodar | How to run

```bash id="vkj12d"
git clone https://github.com/rmgiust/qa-portfolio.git
cd qa-portfolio/selenium-books-scraper
npm install
npx ts-node scrape.ts
```

---

## 📊 Saída | Output

Gera um arquivo:

```bash id="x9gbfp"
books.csv
```

Com:

* Título
* Preço
